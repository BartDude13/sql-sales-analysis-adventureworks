## Dataset

This project uses the **AdventureWorks 2017** sample database provided by Microsoft.

The AdventureWorks dataset represents a fictional global bicycle manufacturing and retail company and includes data related to:
- Sales transactions
- Customers
- Products and product categories
- Geographic and territory information

The dataset is provided as a relational database and is not included in this repository.

**Source:** AdventureWorks 2017 Sample Database (Microsoft)
